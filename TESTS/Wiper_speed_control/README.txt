README - motor controller test

This program was to test and validate the speed control aspect of the greater master program and is trigger by pressing not more than one of the limit switches. 

This program allows for the tuning of the p-control motor controller. 

BE AWARE THIS MOTOR WILL BE CAPABLE OF SUPPLYING A LARGE CURRENT IN THIS INSTANCE.

DO NOT COPY THIS PROGRAM INTO ANOTHER WITHOUT UNDERSTANDING THE FUNCTION OF EACH COMPONENT AND HOW IT RUNS
